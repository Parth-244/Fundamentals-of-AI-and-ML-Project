#Tic Tac Toe Game using Artificial Intelligence
Project Overview

This project is a Python-based Tic Tac Toe game where a human player competes against an AI opponent. The AI uses the Minimax algorithm to make optimal decisions, which means it always tries to play the best possible move.

#Requirements

You need Python 3 installed on your system. No additional libraries are required.

#Setup Instructions

Download or clone this repository to your system. Open the project folder in any code editor such as Visual Studio Code. Make sure the Python file is saved with a .py extension.

#How to Run

Open the terminal and navigate to the project folder. Run the program using:

python filename.py

Replace "filename.py" with your actual file name. Make sure you run the program in the terminal because the game requires user input.

#How to Play

The board displays positions from 1 to 9. You will play as X and the AI will play as O.
Enter a number between 1 and 9 to place your move. The AI will automatically respond after your turn.
The game continues until someone wins or the board is full.

#Features

AI opponent using Minimax algorithm
User input with validation
Clear board display with position guide

#Purpose

The aim of this project is to demonstrate how basic Artificial Intelligence techniques can be applied to a simple game using decision-making logic.
