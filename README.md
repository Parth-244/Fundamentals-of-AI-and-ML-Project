# Tic Tac Toe Game using Artificial Intelligence

## Project Overview  
This project is a Python-based Tic Tac Toe game where a human player plays against an AI.

## Requirements  
You need Python 3 installed.

## How to Run  
Open terminal and run:

python filename.py

## How to Play  
Enter numbers from 1 to 9 to play your move.

## Features  
- AI opponent  
- Input system  
- Simple board display
